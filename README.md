# asdf-swift

Swift plugin for https://github.com/asdf-vm/asdf/

This is Swift plugin for asdf version manager.

# Installation

```bash
asdf plugin-add https://github.com/drujensen/asdf-swift.git
asdf install swift 5.5.2
swift --version
```
